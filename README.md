# PairGS Project Page

This folder contains the static project page for:

**PairGS: Relation-Centric Open-Vocabulary 3D Gaussian Segmentation**

## Preview

Open `index.html` directly in a browser. If a browser blocks local video playback, run a small local server from this directory:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Assets

- Paper link: temporary placeholder until the arXiv link is available.
- PairGS videos: `static/videos/`
- Main page: `index.html`
- Styles: `static/css/index.css`
- Small navbar script: `static/js/index.js`

## Before Publishing

Update `index.html` when these are ready:

- Conference or venue line, if applicable.
- arXiv link, if the paper is public on arXiv.
- Code link, if the implementation will be released.
- Final BibTeX entry after publication.

The Paper and Code buttons are temporary placeholders until the arXiv and code links are available.
